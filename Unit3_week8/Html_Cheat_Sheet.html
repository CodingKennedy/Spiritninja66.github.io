Arrays!

An array is a data object that basically stores an ordered list of values.
An array uses these brackets [ ]. Array_of_intergers[1,2,3,4]

Hashes!

Hashes are pretty simple to use. They are similar to an array, but they require a key. So this will work very much like an index.
The hash will use these ugly brackets { } or curly brackets.
The code would look something like this: grades = { "Jane Doe" => 10, "Jim Doe" => 6 }

Strings!

String are very simple. Anything that is placed between single ' ' or double quotations " " will become a string.
"Dog", "Cat", "Elephant"

Intergers!

These are super simple as well. Think of these as numbers. These we can manipulate and change.
5+5 will return 10.
9-7 will return 2.
